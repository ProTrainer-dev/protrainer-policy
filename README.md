# Privacy Policy / プライバシーポリシー

## 1. About User Data (User Audio Data)
ProTrainer English uses the microphone to collect audio data for the purpose of English pronunciation analysis. This data is processed in real-time to provide feedback to the user and is not stored on our servers or shared with third parties for any other purposes.

**（ユーザーデータ・音声データについて）**
ProTrainer Englishは、英語の発音分析を目的としてマイクから音声データを収集します。このデータはユーザーにフィードバックを提供するためにリアルタイムで処理されます。当方のサーバーに保存されたり、他の目的で第三者に共有されたりすることはありません。

## 2. Data Collection and Usage
We do not collect any personally identifiable information (PII). We may use basic, non-identifying analytical data to improve app performance and user experience.

**（データの収集と利用）**
当アプリは、個人を特定できる情報（氏名、メールアドレス等）を収集しません。アプリのパフォーマンス向上やユーザー体験の改善のため、個人を特定しない基本的な分析データを使用する場合があります。

## 3. Third-Party Services
This app does not use third-party advertising or data-tracking SDKs that share your personal information.

**（第三者サービス）**
当アプリは、個人の情報を共有するような第三者の広告やデータトラッキングSDKを使用していません。
